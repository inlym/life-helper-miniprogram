# life-helper-miniprogram

「小鸣助手」小程序端部分代码

## 目录结构

```bash
life-helper-miniprogram
├── app
│   ├── config  # 配置文件目录
│   │   └── config.prod.js  # 生产环境配置文件
│   │   └── config.ts  # 本地开发、测试环境配置文件
│   └── lib  # 存放预备挂载到 app 实例上文件
│   └── static  # 静态文件目录
│       └── css  # 样式文件目录
│       └── image  # 图片文件目录
├── components  # 自定义组件目录

```
